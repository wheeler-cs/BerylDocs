Beryl ROM Hack Documentation

Welcome to the documentation for the Beryl ROM hack of Pokemon Emerald! This website is designed to provide a (hopefully) comprehensive walkthrough of the modifications that have been made to the base game, including changes to the world, game mechanics, and anything else that affects gameplay.

The side navbar can be used to quickly access central pages of the project.

  • Walkthrough outlines the various areas in the game. No story walkthrough is provided.
  • Technical Changes describes various changes that have been made to the engine.
  • Acknowledgements contains information about the project itself, along with some legal disclaimers.
