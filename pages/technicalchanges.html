Technical Changes

Minor Changes

Small changes to the code base that do not greatly impact gameplay.

  • Maximum display value of in game clock modified to be 9999:99. This does not change the way time is tracked in game.
  • Pokemon hatch from eggs at level 1.

Highly Technical Changes

  • CPU now goes into low power mode while waiting for Vblank. This makes the game run much more efficient. This change breaks compatibility with vanilla.
  • Maps now use Poryscript to define a high-level interface for writing map scripts.

Debugging

  • Added custom debugging maps to assist in debugging the game. Entering the area requires the DEBUGGER item, which is available in the player's PC when the ANNA_DEBUG flag is defined in config.h.
