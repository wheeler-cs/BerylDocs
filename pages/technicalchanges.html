Walkthrough Technical Changes Acknowledgements
Technical Changes

Minor Changes

Small changes to the code base that do not greatly impact gameplay.

  • Maximum display value of in game clock modified to be 9999:99. This does not change the way time is tracked in game.

Item Changes

Addition of new items or modification of existing items.

  • Added the link stone, which allows for obtaining trade evolutions without trading.
  • Fast Ball: Improved catch rate for faster Pokemon.
  • Level Ball: Improved catch rate if target is below current Pokemon's level.
  • Lure Ball: Improved catch rate for fishing encounters.
  • Heavy Ball: Improved catch rate for heavier targets. Reduced catch rate for lighter targets.
  • Love Ball: Improved catch rate for targets of the opposite gender to current Pokemon.
  • Friend Ball: Increases happiness upon capture.
  • Moon Ball: Improved catch rate for Pokemon that evolved using the moon stone.
  • Sport Ball: Standard catch rate.
  • Dusk Ball: Improved catch rate underground or at night.
  • Heal Ball: Heals Pokemon upon capture.
  • Quick Ball: Improved catch rate on first turn.
  • Cherish Ball: Used for distributing special Pokemon.
